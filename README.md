
# 🩺 SmartSaathi – Secure Medical Ecosystem

SmartSaathi is a lightweight emergency medical web application designed to help users store critical health information and quickly contact guardians during emergencies.

The application allows users to register their medical profile, set a secret voice key, and trigger an SOS alert when needed.

---

## 🚀 Features

- 👤 User Identity Registration
- 🏥 Medical Profile Storage
- 🎤 Voice Key Security
- 📞 Emergency Guardian Contacts
- 🚨 One-Tap SOS Alert
- 📍 Live GPS Location Tracking
- ❤️ Simulated Health Vitals
- 🎵 Smart Music Search
- ⚙️ Profile Settings

---

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- LocalStorage API
- Geolocation API
- Font Awesome Icons

---



## ⚙️ How the Application Works

1. User enters **identity details**
2. User adds **medical conditions and allergies**
3. User sets a **secret voice key**
4. User adds **guardian phone numbers**
5. All data is saved in **browser local storage**
6. Dashboard displays:
   - Health summary
   - Simulated heart rate
   - GPS location
7. User can press **SOS button** to call emergency contact.

---

## 🚨 Emergency SOS System

When the SOS button is pressed:

- Alert mode is activated
- Primary guardian is called
- User location is tracked
- Emergency status is displayed
